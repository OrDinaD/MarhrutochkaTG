# 🚌 MarhrutochkaTG - Telegram Бот для мониторинга маршруток

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![Telegram Bot API](https://img.shields.io/badge/Telegram%20Bot%20API-Latest-blue)](https://core.telegram.org/bots/api)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**Современный и надежный Telegram-бот для мониторинга и бронирования рейсов** на направлении Минск ⇄ Островец через [билет.маршруточка.бел](https://билет.маршруточка.бел).

## 🎯 Основные возможности

- **🔍 Продвинутый поиск и мониторинг**: Поиск рейсов по дате, направлению и времени. Автоматический мониторинг с уведомлениями в реальном времени при появлении свободных мест.
- **🔐 Безопасная веб-авторизация**: Вход в личный кабинет сайта, просмотр профиля и истории бронирований.
- **🛡️ Система самовосстановления**: Автоматическое обнаружение, диагностика и исправление сбоев в работе, включая проблемы с сетью, зависимостями и файлами.
- **🔒 Повышенная безопасность**: Встроенные механизмы валидации и очистки пользовательского ввода для защиты от базовых уязвимостей.
- **👑 Админ-панель**: Встроенные инструменты для управления ботом и мониторинга его состояния.
- **🤖 Автобронирование**: Экспериментальная функция для автоматического бронирования мест.

## 🚀 Быстрый старт

### 1. Клонирование и установка

```bash
git clone https://github.com/OrDinaD/MarhrutochkaTG.git
cd MarhrutochkaTG

# Создание и активация виртуального окружения
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка окружения

Создайте файл `.env` по примеру `.env.example` и укажите ваш токен:
```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
ADMIN_TELEGRAM_ID=your_admin_user_id
```
- `TELEGRAM_BOT_TOKEN` можно получить у [@BotFather](https://t.me/BotFather).
- `ADMIN_TELEGRAM_ID` - ваш числовой ID в Telegram, необходим для доступа к админ-панели.

### 3. Запуск бота

```bash
python main.py
```
Этот скрипт запускает и веб-сервер для health-чеков, и самого бота в фоновом режиме.
