# 🚌 MarhrutochkaTG Bot

Telegram-бот для мониторинга и бронирования билетов на маршрутки Минск ↔ Островец.

## 🚀 Функциональность

- 🔍 **Поиск рейсов** с удобными фильтрами
- ⏰ **Мониторинг маршрутов** с уведомлениями о появлении мест
- 🎫 **Бронирование билетов** через интеграцию с официальным сайтом
- 👤 **Управление профилем** и истории поездок
- 🔐 **Безопасная авторизация** на сайте marshrutochka.by
- 📊 **Подробная статистика** и логирование

## 🛠️ Установка

### Локальная разработка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/OrDinaD/MarhrutochkaTG.git
   cd MarhrutochkaTG
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   # или
   venv\Scripts\activate     # Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения:**
   ```bash
   cp .env.example .env
   # Отредактируйте .env файл, добавьте ваш BOT_TOKEN
   ```

5. **Запустите бота:**
   ```bash
   python main.py
   ```

### Деплой на Railway

1. **Форкните репозиторий** на GitHub
2. **Подключите Railway** к вашему GitHub репозиторию
3. **Настройте переменные окружения** в Railway:
   - `BOT_TOKEN` - токен вашего Telegram бота
   - `ADMIN_TELEGRAM_ID` - ваш Telegram ID для админки
4. **Деплой произойдет автоматически**

## 🔧 Конфигурация

### Основные переменные

- `BOT_TOKEN` - токен Telegram бота
- `ADMIN_TELEGRAM_ID` - ID администратора
- `LOG_LEVEL` - уровень логирования (INFO, DEBUG, ERROR)
- `DEVELOPMENT_MODE` - режим разработки

### Настройки мониторинга

- `MONITORING_INTERVAL` - интервал проверки (секунды)
- `ALERT_THRESHOLD_SEATS` - минимальное количество мест для уведомления

## 🤖 Команды бота

- `/start` - запуск бота и главное меню
