# 🚌 MarhrutochkaTG - Telegram Bot для мониторинга маршруток

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![Telegram Bot API](https://img.shields.io/badge/Telegram%20Bot%20API-Latest-blue)](https://core.telegram.org/bots/api)
[![Railway](https://img.shields.io/badge/Railway-Deployment-blueviolet)](https://railway.app)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**Telegram-бот для поиска рейсов и работы с аккаунтом** на сайте [билет.маршруточка.бел](https://билет.маршруточка.бел)

## 🎯 Основные возможности

### 🔍 Поиск рейсов
- **Быстрый поиск** - поиск доступных рейсов по дате
- **Актуальная информация** - данные напрямую с сайта
- **Удобное отображение** - количество мест, время, перевозчик

### 🔔 Мониторинг свободных мест
- **Автоматическая проверка** - каждые 5 минут
- **Настраиваемые параметры** - дата, направление, время
- **Мгновенные уведомления** - при появлении мест

### 🔐 Авторизация через Requests API
- **Быстрый вход** - авторизация через requests без браузера
- **Просмотр профиля** - ФИО, телефон, email, дата рождения
- **Список билетов** - все активные бронирования
- **Безопасность** - сессии хранятся локально

### 🎨 Интерфейс
- **Современный дизайн** - инлайн-кнопки и красивое меню
- **Динамическое меню** - кнопки меняются в зависимости от авторизации
- **Эмодзи** - наглядное отображение информации

## 🚀 Быстрый старт для локальной разработки

### 1. Клонирование репозитория
```bash
git clone [URL репозитория]
cd Marha
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
LOG_BOT_TOKEN=токен_бота_для_логов
LOG_CHAT_ID=id_чата_для_логов
```

### 4. Запуск бота
```bash
python src/bot.py
```

## 🚀 Деплой на Railway

Для деплоя на Railway и настройки системы логирования смотрите [DEPLOY.md](DEPLOY.md).

### Краткая инструкция:

1. Установите Railway CLI:
```bash
npm i -g @railway/cli
```

2. Авторизуйтесь:
```bash
railway login
```

3. Запустите скрипт деплоя:
```bash
./deploy_to_railway.sh
```

## 📊 Мониторинг и управление

### Просмотр логов
```bash
./monitor_logs.sh
```

### Управление ботом на Railway
```bash
./manage_bot.sh
```
