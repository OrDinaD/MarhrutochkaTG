## Структура сайта

### Найденные формы:

1. **Форма регистрации** (`/auth/register`)
   - Поля: Имя, Email, Телефон, Пароль, Подтверждение
   - Method: POST

2. **Форма авторизации** (`/auth/login`) ⭐
   - Поле телефона: `input[name="phone"]` с placeholder "Телефон"
   - Поле пароля: `input[name="password"]` с placeholder "Пароль"
   - Кнопка submit: `input[type="submit"]` с классом `enterButton`
   - Method: POST

3. **Форма восстановления пароля** (`/auth/forget`)
   - Поле телефона для восстановления

4. **Форма поиска маршрутов** (`/schedules`)
   - Поля: places, date
   - Method: GET

5. **Форма проверки статуса бронирования**
   - Поле номера бронирования: `input[name="order-slug"]`
   - Поле последних 4 цифр телефона
