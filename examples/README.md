# 🌐 WebApp интеграция с параметрами

## Краткое описание

Telegram бот теперь передает параметры маршрута и даты при открытии сайта через WebApp. Это позволяет автоматически заполнять форму поиска на сайте.

## Как это работает

### 1. В боте

Когда пользователь:
- Ищет рейсы и нажимает "Открыть сайт бронирования"
- Получает уведомление о найденных рейсах
- Нажимает кнопку "🌐 Открыть сайт"

Бот открывает URL с параметрами в hash:
```
https://билет.маршруточка.бел/#from=minsk&to=ostrovets&date=2025-11-25
```

### 2. На сайте

Сайт читает параметры через JavaScript:

```javascript
const hash = window.location.hash.substring(1);
if (hash) {
    const params = new URLSearchParams(hash);
    const from = params.get('from');      // 'minsk'
    const to = params.get('to');          // 'ostrovets'
    const date = params.get('date');      // '2025-11-25'
    
    // Автозаполнение формы и запуск поиска
    if (from && to) {
        selectRoute(from, to);
        if (date) setDate(date);
        searchRoutes();
    }
}
```

## Формат параметров

### Направления
- `from=minsk&to=ostrovets` - Минск → Островец
- `from=ostrovets&to=minsk` - Островец → Минск
- `from=minsk&to=smorgon` - Минск → Сморгонь
- `from=smorgon&to=minsk` - Сморгонь → Минск
- `from=ostrovets&to=smorgon` - Островец → Сморгонь
- `from=smorgon&to=ostrovets` - Сморгонь → Островец

### Дата
- `date=YYYY-MM-DD` - Формат ISO (например: `2025-11-25`)

## Примеры

1. **С полными параметрами:**
   ```
   https://билет.маршруточка.бел/#from=minsk&to=ostrovets&date=2025-11-25
   ```

2. **Только направление:**
   ```
   https://билет.маршруточка.бел/#from=ostrovets&to=minsk
   ```

3. **Без параметров:**
   ```
   https://билет.маршруточка.бел/
   ```

## Преимущества

✅ Мгновенное автозаполнение формы  
✅ Не требует серверной поддержки (работает на клиенте)  
✅ Совместимо с SPA фреймворками  
✅ Обратная совместимость (работает без параметров)  
✅ Улучшенный UX для пользователей бота

## Реализация на сайте

Для интеграции на вашем сайте:

1. Добавьте код чтения параметров (см. примеры выше)
2. Маппируйте английские названия на локальные
3. Заполните форму поиска
4. Опционально: запустите поиск автоматически

См. полные примеры в `examples/webapp_params_demo.html` и `docs/WEBAPP_INTEGRATION.md`.
