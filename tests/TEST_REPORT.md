### ❌ **Проблемы до оптимизации:**

1. **test_all.py** - 701 строка монолитного unittest кода
2. **Смешение подходов** - unittest + pytest без системы
3. **Отсутствие fixtures** - нет переиспользуемой инфраструктуры
4. **Проблемы с импортами** - тесты не могли импортировать модули
