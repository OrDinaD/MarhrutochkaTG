### ✅ 4. Создание модуля авторизации
**Файл:** `src/auth_manager.py`

**Основные функции:**
- `login(phone, password)` - авторизация через веб-форму
- `get_profile_info()` - получение информации профиля
- `get_bookings()` - получение списка бронирований
- `search_routes(query)` - поиск маршрутов
- `check_booking_status(booking_number)` - проверка статуса бронирования

**Особенности:**
- Использует Playwright для автоматизации браузера
- Надежная система селекторов с несколькими вариантами
- Обработка ошибок и логирование
- Поддержка headless режима

### ✅ 5. Создание модуля форматирования
**Файл:** `src/ticket_formatter.py`

**Основные функции:**
- `format_ticket(ticket_data)` - красивое отображение билетов
- `format_profile(profile_data)` - форматирование профиля
- `format_bookings(bookings_list)` - форматирование списка броней
- `format_routes(routes_list)` - отображение найденных маршрутов
- `format_booking_status(status_data)` - статус бронирования

**Особенности:**
- Использует Unicode символы и эмодзи
- Красивое разделение секций
- Адаптивное форматирование дат и времени
- Поддержка различных статусов
