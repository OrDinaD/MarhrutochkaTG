## 🔧 Технические детали

### Основные компоненты:
1. **RequestsAuthManager** - синхронный менеджер аутентификации
2. **FinalMarshrutochkaParser** - асинхронный парсер сайта
3. **Telegram Bot** - интерактивный бот с мониторингом
4. **ConversationHandler** - обработка диалогов
5. **Scheduler** - планировщик мониторинга

### Функции бота:
- 🔍 Поиск рейсов по дате
- 🔔 Настройка мониторинга с уведомлениями
- 🔒 Вход в аккаунт через requests API
- 👤 Просмотр профиля пользователя
- 🎫 Просмотр активных билетов
- 📊 Управление мониторингами
