## 📋 Инструкции для пользователей

### Для клонирования и запуска:
```bash
git clone https://github.com/OrDinaD/MarhrutochkaTG.git
cd MarhrutochkaTG
cp .env.example .env
# Отредактируйте .env и укажите ваш TELEGRAM_BOT_TOKEN
./scripts/start_bot.sh
```

### Для разработчиков:
```bash
git clone https://github.com/OrDinaD/MarhrutochkaTG.git
cd MarhrutochkaTG
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python tests/test_all.py  # Запуск тестов
```

- ✅ **Уведомления** - мгновенные сообщения при появлении мест
