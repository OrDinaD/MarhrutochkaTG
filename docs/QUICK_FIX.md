# 🚀 БЫСТРАЯ ИНСТРУКЦИЯ - ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС

## ❌ Проблема
Мониторинги слетают после каждого деплоя.

## ✅ Решение (2 минуты)

### 1. Открой Railway Dashboard

Перейди в свой проект MarhrutochkaTG на Railway.

### 2. Добавь одну переменную

**Settings → Variables → New Variable**

```
Variable name:  MONITORING_STORAGE
Value:          redis
```

Нажми **Add** → Railway автоматически переделоплоит бот.

### 3. Готово! ✨

Теперь мониторинги сохраняются в Redis и **НЕ пропадают** при деплое.

---

## 🔍 Как проверить что работает

1. Открой бота в Telegram
2. `/start` → "🔔 Настроить мониторинг"
3. Настрой любой мониторинг
4. Сделай `git commit` и `git push` (любое изменение)
5. Подожди ~1 минуту пока Railway переделоплоит
6. Открой бота → "📊 Мои мониторинги"
7. **Мониторинг на месте?** ✅ Всё работает!
