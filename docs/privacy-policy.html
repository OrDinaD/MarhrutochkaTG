🔒 Эта политика описывает, как мы обрабатываем ваши данные при использовании Telegram-бота "Мониторинг маршруточки" (@MarshrutochkaOst_bot).

Какие данные мы собираем

  • Идентификатор пользователя Telegram - для персонализации сервиса и идентификации
  • Предпочтения по маршрутам - выбранные направления и предпочитаемое время поездок
  • Настройки уведомлений - частота уведомлений и типы push-сообщений
  • История взаимодействий - команды, запросы поиска для улучшения алгоритмов
  • Временные метки активности - время последнего использования бота

Как мы используем ваши данные

  • Отправка уведомлений - информирование о свободных местах каждые 5 минут в режиме реального времени
  • Персонализация поиска - показ наиболее релевантных маршрутов на основе ваших предпочтений
  • Улучшение алгоритмов - анализ популярных направлений и времени для оптимизации
  • Техническая поддержка - диагностика и решение проблем с работой бота
  • Аналитика сервиса - понимание паттернов использования для улучшения функционала

Безопасность данных

🛡️ Гарантии безопасности

  • 🚫 НЕ передаем персональные данные третьим лицам ни при каких обстоятельствах
  • � Шифрование - все данные хранятся в зашифрованном виде
  • 🔒 Ограниченный доступ - только авторизованные системы имеют доступ к данным
  • ⏰ Минимальное хранение - данные хранятся только необходимое время
  • 🗑️ Право на удаление - полное удаление всех данных по запросу
  • 🔄 Регулярные бэкапы - защита от потери данных с возможностью восстановления

Ваши права

  • 📋 Доступ к данным - запросить полную информацию о хранимых данных
  • ✏️ Исправление данных - обновить или изменить свои предпочтения
  • 🗑️ Удаление данных - запросить полное и необратимое удаление всех данных
  • 🔕 Отказ от уведомлений - отключить все типы push-уведомлений
  • 📤 Экспорт данных - получить копию ваших данных в машиночитаемом формате
  • ⛔ Ограничение обработки - приостановить обработку данных при сохранении аккаунта

Автоматический сбор данных

  • 📊 Логи взаимодействия - время и частота использования бота для улучшения производительности
  • 🔧 Технические метрики - данные о производительности для диагностики ошибок
  • 📈 Аналитика использования - анонимная статистика для развития функционала
  • 🚨 Журналы ошибок - автоматическая запись ошибок для их устранения

Изменения в политике

Мы можем обновлять эту политику конфиденциальности для отражения изменений в нашей практике или по правовым требованиям. О существенных изменениях мы обязательно уведомим пользователей через бота или на этой странице за 30 дней до вступления изменений в силу.
