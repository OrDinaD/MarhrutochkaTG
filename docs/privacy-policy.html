Политика конфиденциальности

Эта политика описывает, как мы обрабатываем ваши данные при использовании Telegram-бота "Мониторинг маршруточки" (@MarshrutochkaOst_bot).

Какие данные мы собираем

  • Идентификатор пользователя Telegram - для персонализации сервиса
  • Предпочтения по маршрутам - выбранные направления и время поездок
  • Настройки уведомлений - частота и типы уведомлений
  • История поиска - для улучшения рекомендаций

Как мы используем ваши данные

  • Отправка уведомлений - информирование о свободных местах каждые 5 минут
  • Персонализация поиска - показ наиболее релевантных маршрутов
  • Улучшение сервиса - анализ популярных направлений и времени
  • Техническая поддержка - решение проблем с работой бота

Безопасность данных

  • 🔒 Мы НЕ передаем персональные данные третьим лицам
  • 🛡️ Данные хранятся в защищенном виде с ограниченным доступом
  • ⏰ Данные хранятся только необходимое время для работы сервиса
  • 🗑️ Вы можете запросить удаление всех своих данных в любое время

Ваши права

  • Доступ к данным - вы можете запросить информацию о хранимых данных
  • Исправление данных - вы можете обновить свои предпочтения
  • Удаление данных - вы можете запросить полное удаление данных
  • Отказ от уведомлений - вы можете отключить все уведомления

Автоматический сбор данных

  • Логи взаимодействия - время использования бота для статистики
  • Технические данные - для диагностики и исправления ошибок
  • Данные о производительности - для оптимизации работы

Изменения в политике

Мы можем обновлять эту политику конфиденциальности. О существенных изменениях мы уведомим через бота или на этой странице.
