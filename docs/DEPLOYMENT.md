# Инструкция по развертыванию Telegram-бота маршруточки

## ✅ ПРОЕКТ ГОТОВ К ИСПОЛЬЗОВАНИЮ!

Telegram-бот для мониторинга рейсов маршруточки Минск-Островец успешно разработан и протестирован.

## 🎯 Что реализовано

### 🔍 Парсер сайта маршруточки
- ✅ Асинхронный парсер с использованием aiohttp + BeautifulSoup
- ✅ Получение расписания через API /schedules
- ✅ Извлечение всех ключевых данных: время, места, перевозчик, длительность
- ✅ Поддержка обеих направлений: Минск ⇄ Островец
- ✅ Обработка ошибок и валидация данных

### 🤖 Telegram-бот
- ✅ Полнофункциональный бот на python-telegram-bot 22.2
- ✅ Справка и помощь: /start, /help
- ✅ Настройка мониторинга: /monitor
- ✅ Управление мониторингами: /monitoring
- ✅ Красивое форматирование сообщений с эмодзи
- ✅ Автоматическое разбиение длинных сообщений

### 🔔 Система мониторинга
- ✅ Автоматический мониторинг каждые 30 минут (настраивается)
- ✅ Умные уведомления только при наличии свободных мест
- ✅ Настраиваемый порог уведомлений (по умолчанию ≥5 мест)
- ✅ Группировка запросов для оптимизации

### 📊 Логирование и мониторинг
- ✅ Полное логирование в файлы с ротацией по дням
- ✅ Graceful shutdown при получении сигналов
- ✅ Обработка всех возможных ошибок
- ✅ Тестовые утилиты для диагностики

## 🚀 Быстрый старт

### 1. Проверка готовности
```bash
cd /Users/vlad/Developer/Marha
source venv/bin/activate

# Проверяем работу бота
python main.py --help
```

### 2. Запуск бота
```bash
# Основной запуск
python main.py

# Или через вспомогательный скрипт
scripts/start_bot.sh

# Фоновый режим
nohup python main.py > bot.log 2>&1 &
```

### 3. Настройка автозапуска (macOS)
Создайте файл `~/Library/LaunchAgents/marshrutochka.bot.plist`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>marshrutochka.bot</string>
    <key>ProgramArguments</key>
    <array>
        <string>/Users/vlad/Developer/Marha/venv/bin/python</string>
        <string>/Users/vlad/Developer/Marha/main.py</string>
    </array>
    <key>WorkingDirectory</key>
    <string>/Users/vlad/Developer/Marha</string>
    <key>RunAtLoad</key>
    <true/>
    <key>KeepAlive</key>
    <true/>
    <key>StandardOutPath</key>
    <string>/Users/vlad/Developer/Marha/logs/launchd.out</string>
    <key>StandardErrorPath</key>
    <string>/Users/vlad/Developer/Marha/logs/launchd.err</string>
</dict>
</plist>
```

Затем:
```bash
launchctl load ~/Library/LaunchAgents/marshrutochka.bot.plist
launchctl start marshrutochka.bot
```
